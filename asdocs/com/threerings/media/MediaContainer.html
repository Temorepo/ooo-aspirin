shutdown()method 
public final function shutdown(completely:Boolean = true):void

Unload the media we're displaying, clean up any resources. This method may not be overridden, as WILL_SHUTDOWN is useless if it doesn't happen first. Subclass doShutdown() instead, or even better, shutdownMedia().

Parameters

completely:Boolean (default = true) — if true, we're going away and should stop everything. Otherwise, we're just loading up new media.
