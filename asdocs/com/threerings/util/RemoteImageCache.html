public function getBitmapData(url:String):BitmapData

Returns the BitmapData for the image associated with the given url, if the image has been loaded. Otherwise, returns null. No load will be initiated as a result of calling this function.

getImage()method 
public function getImage(url:String, retryOnFailure:Boolean = false):Sprite

If the image at the given url has been loaded, returns a Sprite containing a single Bitmap child with the contents of that image. If the image has not been loaded, a load will be initiated, and the returned Sprite will have the image added to it when the image has completed loading.

Parameters

url:String — if the image isn't already loaded, and it fails to load, retry until it succeeds, or until there's an unrecoverable failure.
 
retryOnFailure:Boolean (default = false)

Returns
