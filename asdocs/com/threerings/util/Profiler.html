Packagecom.threerings.util
Classpublic class Profiler

Modified version of the PBE Profiler to fit OOO logging. Simple, static hierarchical block profiler. Set enabled to true to start profiling and call report to print the results. Use it by calling Profiler.enter("CodeSectionName"); before the code you wish to measure, and Profiler.exit("CodeSectionName"); afterwards. Note that Enter/Exit calls must be matched - and if there are any branches, like an early return; statement, you will need to add a call to Profiler.exit() before the return. Min/Max/Average times are reported in milliseconds, while total and non-sub times (times including children and excluding children respectively) are reported in percentages of total observed time.
