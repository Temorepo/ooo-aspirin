Packagecom.threerings.util
Classpublic class Ordering
InheritanceOrdering Object

Implements a sort function based on position in a fixed backing array. This is equivalent to using indexOf(a) - indexOf(b) but with dictionary lookup performance instead of linear. This is intended for arrays that do not change often, but could be adapted by exposing specific mutators like push and splice that only update the necessary indices.
