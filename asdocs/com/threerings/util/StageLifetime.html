listen()method
public static function listen(disp:DisplayObject, onAdd:Function, onRemove:Function, sync:Boolean = false):void

Adds the given handlers for ADDED_TO_STAGE and REMOVED_FROM_STAGE events coming from the given disp, but guarantee the call order will always be interleaved. The first listener added will be onAdd if disp.stage is null, otherwise onRemove.

Parameters

disp:DisplayObject
 
onAdd:Function
 
onRemove:Function
 
sync:Boolean (default = false) — if passed and set to true then the onAdd or addRemove callback will be called initially depending on whether the given object is already on stage. A null event is used in this case.
